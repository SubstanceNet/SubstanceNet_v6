## Від класифікації до біологічного розпізнавання

## 1. Мета сесії

Реалізувати парадигму впізнавання замість класичного train→classify. Гіпотеза: SubstanceNet як архітектура свідомості не потребує backpropagation для розпізнавання — достатньо зберегти приклади в епізодичній пам'яті та впізнавати через порівняння.

Теоретична основа:
- Complementary Learning Systems (McClelland et al., 1995; Kumaran, Hassabis, McClelland, 2016)
- Тензори остенсивних визначень (Дубовиков)
- Зорова кора вроджена (Hubel & Wiesel, 1962)

## 2. Кольоровий V1 — RetinalLayer

### Реалізація

Додано `RetinalLayer` в `src/cortex/v1.py` — моделює 4 типи фоторецепторів сітківки:

| Фоторецептор | Канал | Спектральна чутливість |
|---|---|---|
| Палички (rods) | Яскравість | 0.2126R + 0.7152G + 0.0722B |
| L-колбочки | Червоний | 0.70R + 0.30G |
| M-колбочки | Зелений | 0.20R + 0.70G + 0.10B |
| S-колбочки | Синій | 0.10G + 0.90B |

RGB → 4 канали через фіксовану спектральну матрицю (CIE cone fundamentals), потім center-surround антагонізм (як в гангліозних клітинах сітківки).

`GaborFilterBank` оновлено з параметром `in_channels` — кожний Gabor-фільтр реплікується по всіх вхідних каналах.

### Результати

| Вхід | CIFAR-10 (1 epoch) | R |
|---|---|---|
| Grayscale (1 канал) | 36.36% | 0.4098 |
| **RGB (4 канали retina)** | **40.56%** | **0.4098** |

Кольоровий вхід дає +4.2%. Backward-сумісність з grayscale збережена. MNIST 1-epoch не перевірявся (залишається 95.94%).

## 3. Парадигма впізнавання — Експерименти

### 3.1 Ідея

Замість: `train(backprop, 60000) → classify(logits)`
Тестуємо: `encode(few examples) → store(memory) → recognize(similarity)`

Модель **НЕ тренована** — випадкові ваги + біологічні Gabor-фільтри. Жодного gradient descent.

### 3.2 Експеримент 1: Abstract vs Features

Перший тест використовував abstract representation (dim=3) для порівняння. Результат — 8.4%, рівень випадковості. Причина: abstract з випадковими вагами не розрізняє класи (separation ratio = 1.6).

Переключення на amplitude+phase features (dim=128) змінило результат радикально:

| Representation | Dim | Separation ratio | Recognition acc |
|---|---|---|---|
| abstract | 3 | 1.6 | 8.4% |
| **amplitude+phase** | **128** | **52.5 / 40.9** | **42.9%** |

Висновок: для впізнавання потрібні багатовимірні ознаки, а не стиснений abstract.

### 3.3 Експеримент 2: Масштабування N-shot

Скільки прикладів потрібно для впізнавання? (cosine similarity, top-5 voting)

| Приклади/клас | Всього | Accuracy | ×Random |
|---|---|---|---|
| 1 | 10 | 8.7% | 0.9× |
| 5 | 50 | 46.0% | 4.6× |
| 10 | 100 | 51.2% | 5.1× |
| 20 | 200 | 56.2% | 5.6× |
| 50 | 500 | 64.3% | 6.4× |
| 100 | 1000 | **71.9%** | **7.2×** |
| 60000 (backprop) | 60000 | 95.9% | 9.6× |

Крива росте логарифмічно — кожне подвоєння прикладів дає +5-8%. Це відповідає біологічному навчанню: більше досвіду → краще впізнавання, але не лінійно.

### 3.4 Експеримент 3: Повний цикл навчання

Повторення → короткострокова пам'ять → консолідація ("сон") → впізнавання через прототипи.

Параметри: 20 прикладів/клас, 5 повторень кожного.

| Метод | Accuracy | Об'єктів у пам'яті | Стиснення |
|---|---|---|---|
| Сирі епізоди (short-term) | 44.7% | 200 | 1× |
| **Прототипи (consolidated)** | **46.3%** | **10** | **20×** |

Консолідація працює: **10 прототипів краще ніж 200 сирих епізодів**. Усереднення прибирає шум, залишає суттєве. Пам'ять стиснута в 20 разів.

Повний біологічний цикл:
```
Повторення (5×)          → hippocampus short_term_buffer (1000 епізодів)
Консолідація ("сон")     → prototypes (10 прототипів)
Впізнавання              → cosine similarity з прототипами
```

### 3.5 Експеримент 4: Якість ознак по стадіях зорового тракту

Яка стадія дає найкращі ознаки для впізнавання? (20-shot, прототипи)

| Стадія | Accuracy | Тип операцій | "Вродженість" |
|---|---|---|---|
| after_wave (V1) | 43.7% | Gabor фільтри | ✅ вроджена |
| after_nonlocal | 43.6% | Attention | ❌ випадкові ваги |
| **after_v2** | **53.7%** | **FFT + різниці** | **✅ вроджена** |
| after_v3 | 54.7% | Cross-stream gating | ⚠️ частково |
| after_v4 | 53.2% | Multi-scale attention | ❌ випадкові, шкодять |

Ключові знахідки:

1. **V2 (MosaicField18) дає +10%** — найбільший стрибок якості. Операції V2 (FFT, roll-diff) фіксовані, не потребують навчання — як і V1 Gabor.

2. **V4 з випадковими вагами шкодить** (-1.5% відносно V3). Випадкова attention додає шум замість корисних ознак.

3. **"Вроджені" шари (V1+V2) = 53.7%** без тренування. Це відповідає біології: базова зорова обробка генетично запрограмована (Hubel & Wiesel, 1962, 1968).

## 4. Теоретичне значення

### Відповідність CLS теорії

| CLS компонент | Біологія | SubstanceNet |
|---|---|---|
| Швидке запам'ятовування | Гіппокамп | hippocampus.encode_and_store() |
| Короткострокова пам'ять | CA1/CA3 | short_term_buffer |
| Консолідація (replay/сон) | Hippocampal replay | hippocampus.consolidate() |
| Довгострокова пам'ять | Неокортекс (прототипи) | prototypes |
| Впізнавання | Pattern completion | retrieve_similar() / cosine sim |

### Вроджене vs набуте

| Компонент | Біологія | SubstanceNet | Потребує навчання? |
|---|---|---|---|
| V1 (Gabor) | Генетично запрограмована | Фіксовані фільтри | **НІ** |
| V2 (stripes) | Генетично запрограмована | FFT + diff (фіксовані) | **НІ** |
| V3 (dynamic form) | Частково вроджена | Gating (випадкові) | ТАК |
| V4 (objects) | Формується досвідом | Attention (випадкові) | ТАК |
| Hippocampus | Вроджена структура | Grid/Place/Time cells | **НІ** (структура) |
| Consciousness | Вроджений механізм | ψ_C = F[P̂[ψ_C]] | **НІ** (R-targeting) |

### Зв'язок з Дубовиковим

Тензори остенсивних визначень (T = 2^D): фундаментальні ознаки об'єкта формують повну групу для ідентифікації **без тренування**. В SubstanceNet:
- D = ознаки з V1→V2 (Gabor + FFT + diff) — "фундаментальні ознаки"
- T = amplitude+phase representation (128-dim) — "повна група ознак"
- Ідентифікація = cosine similarity з базою — "порівняння з базою"

## 5. Порівняння парадигм

| Параметр | Backprop (CNN) | Recognition (SubstanceNet) |
|---|---|---|
| Навчальні дані | 60000 зображень | 100–1000 зображень |
| Метод навчання | Gradient descent | Запам'ятовування + консолідація |
| Що зберігається | Ваги мережі | Епізоди / прототипи |
| Час навчання | Хвилини–години | Мілісекунди (один прохід) |
| Accuracy (MNIST) | **95.9%** | **71.9%** (100-shot, без тренування) |
| Пояснюваність | Black box | Показує найбільш схожий приклад |
| Нові класи | Перетренувати | Додати приклади |
| Біологічність | Собака Павлова | Людське впізнавання |

## 7. Наступні кроки

1. **Інтегрувати recognition в Hippocampus** — зараз cosine similarity реалізований поза hippocampus. Потрібно додати features-based storage та prototype-based recognition як методи моделі.

2. **V3/V4 "дозрівання"** — реалізувати біологічно обґрунтоване формування V3/V4 через досвід (unsupervised learning на прикладах), а не backprop з labels.

3. **CIFAR-10 recognition** — перевірити парадигму на кольорових зображеннях з RetinalLayer.

4. **Гібридний режим** — комбінація вроджених ознак (V1+V2) з набутими (V3+V4), де набуті формуються через consolidation, а не через gradient descent.

5. **Порівняння з few-shot learning** — протестувати проти стандартних few-shot методів (Matching Networks, Prototypical Networks) для позиціонування.

## 8. Головний висновок

SubstanceNet v4 продемонструвала принципово іншу парадигму розпізнавання зображень:

**Без жодного тренування**, з "вродженими" Gabor-фільтрами (V1) та FFT/diff операціями (V2), система досягає 53.7% accuracy на MNIST через порівняння з 10 прототипами. З 1000 збережених прикладів — 71.9%.

Це не конкурент CNN по accuracy. Це **інший підхід до інтелекту**: не статистичне навчання на мільйонах прикладів, а впізнавання через порівняння з досвідом. Як людина, яка бачила 100 цифр і впізнає нову — не тому що пройшла 60000 ітерацій gradient descent, а тому що пам'ятає схожі.

Consciousness (R ≈ 0.41, κ ≈ 1) стабільно працює в обох парадигмах — і при backprop, і при recognition.

---

*Парадигма впізнавання підтверджена як працюючий прототип. "Вроджені" зорові шари V1+V2 дають біологічно обґрунтоване розпізнавання без тренування.*
