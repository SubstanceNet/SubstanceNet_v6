## Зміни цієї сесії

### 1. RetinalLayer — кольоровий вхід

Додано модель сітківки в `src/cortex/v1.py`: RGB → 4 фоторецепторних канали (палички + L/M/S колбочки) через фіксовану спектральну матрицю + center-surround антагонізм. GaborFilterBank приймає `in_channels` параметр. SubstanceNet приймає `in_channels=1|3`.

Результат: CIFAR-10 RGB 40.56% vs grayscale 36.36% (+4.2%). Backward-сумісність з MNIST збережена.

### 2. Парадигма впізнавання — прототип

Реалізовано `research/recognition_paradigm.py` — encode→store→recognize без backprop. Ключові результати на MNIST з **нетренованою** моделлю:

| Метод | Accuracy | Навчальних прикладів |
|---|---|---|
| 5-shot (cosine sim) | 46.0% | 50 |
| 100-shot (cosine sim) | 71.9% | 1000 |
| Прототипи (consolidated) | 46.3% | 200 → 10 прототипів |
| Backprop (reference) | 95.9% | 60000 |

Консолідація: 10 прототипів (46.3%) > 200 сирих епізодів (44.7%) при 20× стисненні пам'яті.

### 3. Якість "вроджених" ознак по стадіях V1→V4

| Стадія | Accuracy | Вроджена? |
|---|---|---|
| V1 (Gabor) | 43.7% | ✅ так |
| **V2 (FFT + diff)** | **53.7%** | **✅ так** |
| V3 (gating) | 54.7% | ⚠️ частково |
| V4 (attention) | 53.2% | ❌ шкодить |

V1+V2 = 53.7% без тренування. Підтвердження Hubel & Wiesel: базова зорова обробка генетично запрограмована.

### 4. V2 streams interface

`MosaicField18.forward(x, return_streams=True)` тепер повертає окремі потоки `{thick, thin, pale}` для майбутньої фазової інтерференції в V3. Backward-compatible.

## План: Еволюція V3 (послідовність 3→2→1)

### Крок 3 (наступна сесія): Темпоральний вимір

**Мета:** Дати V3 реальний динамічний вхід замість статичних зображень.

**Що робити:**
- Написати генератор синтетичних 2D-примітивів (кола, квадрати, лінії) з відомими векторами швидкості, обертання, масштабу
- Формат: `[B, T, C, H, W]` — послідовність кадрів
- Ground truth для кожного параметру руху (Δx, Δy, Δθ, Δs)
- Верифікація: фазовий зсув thick_stripes ∝ реальному зсуву об'єкта

**Чому власний генератор, а не Moving MNIST:**
Повний контроль над ground truth руху. Точна калібрація фазових зсувів. Можливість ізолювати окремі типи трансформацій.

### Крок 2 (після 3): Фазова інтерференція

**Мета:** Замінити σ(W·concat) на хвильову суперпозицію.

**Формула:**
```
|ψ_form + ψ_motion|² > |ψ_form|² + |ψ_motion|²  при Δφ ≈ 0
```

Якщо контур (pale) і вектор руху (thick) належать одному об'єкту — їхні фази синхронізуються → конструктивна інтерференція → підсилення амплітуди. Це розв'язує binding problem через фізику хвиль.

### Крок 1 (після 2): Фазово-залежна пластичність

**Мета:** Замінити backprop для V3 на локальне Хеббівське правило.

**Формула:**
```
ΔW_ij ∝ ∫ cos(φ_i(t) - φ_j(t)) dt
```

Вага зростає коли фази когерентні в часі — аналог STDP (Spike-Timing-Dependent Plasticity). Це дозволить V3 формувати стійкі репрезентації через досвід, а не через gradient descent.
