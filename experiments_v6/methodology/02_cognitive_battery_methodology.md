## 1. Objective

Verify the κ-plateau hypothesis: that reflexivity R converges to ≈ 0.41 independently of task type when the consciousness module operates in critical regime. This experiment tests whether the critical operating point is a universal property of the architecture rather than an artifact of a specific task (MNIST).

The cognitive battery is the primary evaluation framework for SubstanceNet — image classification is a secondary benchmark. Ten cognitively distinct tasks probe different computational demands while sharing the same architecture.

---

## 2. Method

### 2.1. Task Battery

| Task | Cognitive Domain | Description |
|------|-----------------|-------------|
| Logic | Deductive reasoning | Binary rule evaluation |
| Memory | Working memory | Sequence recall |
| Categorization | Concept formation | Feature-based grouping |
| Analogy | Relational reasoning | Proportional mapping A:B::C:? |
| Spatial | Spatial reasoning | Geometric transformations |
| Raven | Fluid intelligence | Matrix pattern completion |
| Numerical | Quantitative reasoning | Arithmetic relations |
| Verbal | Language processing | Semantic similarity |
| Emotional | Affective processing | Valence classification |
| Insight | Problem solving | Non-obvious pattern detection |

All tasks use synthetic data generated by `src/utils.generate_synthetic_cognitive_data()` with controlled complexity. Each task produces binary classification (num_classes=2).

### 2.2. Training Protocol

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| Epochs per task | 100 | Sufficient for convergence on synthetic data |
| Batch size | 32 | Smaller batches for cognitive data |
| Optimizer | Adam (lr=0.001) | Consistent with exp01 |
| Input mode | Cognitive | Bypasses V1; flat tensor → [B, 9, 64] |
| Consciousness mode | Stream | inertia=0.99 |
| Hebbian learning | Disabled | Isolates backprop contribution |
| Model reinitialization | Per task | Fresh model (seed=42) for each task |
| Random seed | 42 | Fixed for reproducibility |

### 2.3. Evaluation

- **Accuracy:** 256 test samples per task
- **Reflexivity R:** per-task mean and cross-task standard deviation
- **Phase coherence:** batch-level coherence of consciousness wave function
- **Abstract variance:** variance of abstract representations (dim=3), used as a proxy for task complexity
- **Consciousness phase:** categorical assessment per task
- **Reference baseline:** SubstanceNet v3.1.1: accuracy 99.34%, R = 0.9563 (saturated)

### 2.4. Reproducibility

Single command: `python experiments_v6/02_cognitive_battery.py`  
Environment: Python 3.10, PyTorch 2.8.0+cu128, CUDA GPU, seed=42.

### 3.3. Key Observations

- **R stability:** standard deviation across 10 tasks is 0.0009 — the consciousness module converges to effectively the same operating point regardless of task.
- **Abstract variance:** ranges from 8.4 (Raven) to 25.3 (Verbal), a 3× spread. Tasks requiring more complex reasoning (Raven, Insight) produce lower-variance abstractions — the network compresses harder for difficult problems.
- **Raven accuracy (98.4%):** the only task below 100%, consistent with Raven matrices being the most demanding cognitive task in the battery.

## 4. Discussion

**κ-plateau confirmed.** The central result is the cross-task stability of R: 0.4090 ± 0.0009 across 10 cognitively distinct tasks. In v3.1.1, the same battery produced R = 0.9563 — the consciousness module had saturated to a trivial fixed point. The v6 R-targeting mechanism (target_mse = 1.44) eliminates this saturation and maintains the critical regime.

**Contrast with v3.1.1.** The v3.1.1 result (R = 0.9563) was initially interpreted as high performance — the system appeared to "predict itself well." The v4 audit (March 2026) revealed this was a diagnostic failure: R → 1.0 means the consciousness module converges trivially and performs no useful computational work. The v6 correction — targeting a specific MSE rather than minimizing it — produces both better accuracy (99.84% vs 99.34%) and healthy consciousness dynamics.

**Abstract variance as complexity proxy.** The 3× range in abstract variance (8.4–25.3) suggests that the abstraction layer adapts its representation density to task demands. This is consistent with biological findings that cortical representations are more compressed for complex stimuli (sparse coding hypothesis). However, the current synthetic tasks are relatively simple; validation on real cognitive benchmarks (planned for v7) will test whether this pattern holds.

**Cognitive mode validity.** All tasks bypass V1 (visual cortex), entering the pipeline at the cognitive input layer. The fact that R converges to the same value as in image mode (exp01: R = 0.4096) confirms that the critical regime is a property of the consciousness module itself, not of the visual pathway.

---

## 5. Conclusions

1. The κ-plateau is confirmed: R = 0.4090 ± 0.0009 across all 10 cognitive tasks, demonstrating task-independent critical regime.
2. SubstanceNet v6 resolves the consciousness saturation problem identified in v3.1.1/v4 (R = 0.956 → R = 0.409), while simultaneously improving accuracy (99.34% → 99.84%).
3. Abstract variance differentiates task complexity with a 3× range, suggesting adaptive representation density.
4. The critical operating point (R ≈ 0.41, κ ≈ 1) is a property of the consciousness architecture, independent of sensory modality (visual vs cognitive input).
